# 🐧 bài giảng về Cấu trúc Thư mục trong Linux (Linux Directory Structure).

> Bài học này cực kỳ quan trọng cho các Web Developer và Quản trị viên hệ thống để hiểu cách Linux sắp xếp dữ liệu, cấu hình và ứng dụng

---

### 📂 Cấu Trúc Thư Mục Trong Linux & Ứng Dụng

> Trong bài học này, bạn sẽ học về cấu trúc thư mục của Linux, nơi lưu trữ các thành phần khác nhau của hệ điều hành, cũng như cách các ứng dụng bên thứ ba áp dụng cùng một quy tắc cấu trúc thư mục này

### 🤔 Tại sao bạn cần quan tâm đến Cấu trúc Thư mục Linux? 
#### Hiểu rõ cấu trúc thư mục sẽ giúp bạn:

* Nâng cao kiến thức tổng quan về Linux và dễ dàng điều hướng hệ thống tệp tin.

* Tiết kiệm thời gian tìm kiếm vì biết chính xác file hệ thống/ứng dụng nằm ở đâu.ích.

* Hỗ trợ chẩn đoán và khắc phục lỗi (troubleshooting) nhanh chóng bằng cách tra cứu file nhật ký (logs) và file cấu hình (configurations).

* Quản trị và vận hành các ứng dụng web/phần mềm hiệu quả hơn.
